# Custom Markdown Syntax
## Why?
Only the website can complie custom markdown syntax  

## Download Button
This is an custom markdown download button
```
[download(Download)(https://play.google.com/store/apps/details?id=com.dergoogler.hybridtts)]
```

## Badge
This is an Badge img (svg), it works with [badge•net][badge]
```
[badge(Made%20With%20❤️)(In%20VS%20Code%20&%20Github)(green)
```
See an demo [here][demo]
